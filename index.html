Docker

Posted on Jan 29, 2016
Sudah bosan mendengar kata Docker sepanjang tahun 2015? Sayang sekali, karena dari skala hype Fantastic Four sampai Mad Max: Fury Road, tampaknya tahun ini Docker akan ada di tahap Captain America: Civil War. Eric Brewer, engineer dari Google, berkata tentang keberadaan Docker, “teknologi pengembang perangkat lunak belum pernah sepesat dan semasif ini sejak framework Ruby on Rails delapan tahun yang lalu.” Sebelum membahas apa itu Docker, inilah yang biasanya terjadi di proses pengembangan perangkat lunak (software development).
